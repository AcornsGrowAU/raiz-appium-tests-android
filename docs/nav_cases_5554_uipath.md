Derived from `docs/nav_map_5554.md` + page objects + existing tests (no device run).

Conventions:
- Back-stack note follows the map's confirmed rule (mismatch #6): screens reached via the
  Home tab surface (Home / Past=history / Future / performance-day / performance-month)
  EXIT the app to launcher on back; modally-pushed screens (invest/deposit/withdraw/
  recurring/performance/transactions/portfolio) RETURN to Home. Tab-to-tab switching
  stays on Home (no back-stack push). Modal sheets dismiss on back without leaving Home.
- All cases SAFE: no money committed (cancel any confirmation sheet), no logout.
- COVERED cites file::Class::test where it exists.
